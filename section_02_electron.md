│   FOUR COMPONENTS:                                                      │
│                                                                         │
│   1. COIN      — two-sided disc, White and Black                        │
│   2. SPRING    — connects the two sides, same material as disc          │
│   3. MEMBRANE  — liquid film, always opposite color to visible side     │
│   4. TRAPPED PHOTON — lives inside the coin, drives everything          │
│                                                                         │
│   ═══════════════════════════════════════════════════════════════════   │
│                                                                         │
│   THE COIN:                                                             │
│                                                                         │
│   Side A = BLACK FLAT (compressed/hard)                                 │
│   Side A = EXPANDED WHITE (inflated/soft)                               │
│                                                                         │
│   Side B = WHITE FLAT (compressed/hard)                                 │
│   Side B = EXPANDED BLACK (inflated/soft)                               │
│                                                                         │
│   The two sides spin in OPPOSITE DIRECTIONS.                            │
│   This is a coil — each side is a spiral spinning                       │
│   counter to the other.                                                 │
│   Two opposite spins = net spin ½.                                      │
│   THIS is where electron spin ½ comes from.                             │
│   Not a mathematical abstraction — coil geometry.                       │
│                                                                         │
│   ═══════════════════════════════════════════════════════════════════   │
│                                                                         │
│   THE TRAPPED PHOTON — THE ENGINE:                                      │
│                                                                         │
│   Inside the coin lives a trapped photon.                               │
│   It bounces between the two sides.                                     │
│   White side → black side → white side → repeat.                        │
│   Each bounce drives the membrane to flip color.                        │
│   The photon IS the engine of the pump.                                 │
│   Without the trapped photon, there is no pump.                         │
│   Without the pump, there is no electron.                               │
│                                                                         │
│   ─────────────────────────────────────────────────────────────────    │
│                                                                         │
│   UNOBSERVED — PHOTON IN TRANSIT:                                       │
│                                                                         │
│        ╔═══════════════════════╗                                        │
│        ║   WHITE SIDE          ║                                        │
│        ║    ∿∿∿∿∿∿∿∿∿∿∿∿∿      ║  ← photon SPREAD across               │
│        ║   ∿∿∿∿∿∿∿∿∿∿∿∿∿∿∿     ║    both sides simultaneously          │
│        ╠═══════════════════════╣    in transit between bounces          │
│        ║   spring (extended)   ║                                        │
│        ╠═══════════════════════╣                                        │
│        ║   BLACK SIDE          ║                                        │
│        ║    ∿∿∿∿∿∿∿∿∿∿∿∿∿      ║                                        │
│        ╚═══════════════════════╝                                        │
│                                                                         │
│   Photon in transit = superposition of positions                        │
│   Neither fully at white nor fully at black                             │
│   THIS IS SUPERPOSITION                                                 │
│   Not a metaphor. Physical location mid-bounce.                         │
│                                                                         │
│   ─────────────────────────────────────────────────────────────────    │
│                                                                         │
│   OBSERVED — PHOTON PINNED:                                             │
│                                                                         │
│        ╔═══════════╗                                                    │
│        ║  WHITE    ║                                                    │
│        ║     ●     ║  ← photon PINNED to white side                     │
│        ╠═══════════╣  ← spring compressed by observation                │
│        ║  BLACK    ║                                                    │
│        ╚═══════════╝                                                    │
│                                                                         │
│   Compression forces photon to one side.                                │
│   Photon position becomes definite.                                     │
│   THIS IS COLLAPSE.                                                     │
│   Not mystery. Pressure pins the photon.                                │
│                                                                         │
│   ═══════════════════════════════════════════════════════════════════   │
│                                                                         │
│   THE QUANTUM STATE = PHOTON POSITION:                                  │
│                                                                         │
│   Photon near WHITE side  →  spin UP   (+½)                             │
│   Photon near BLACK side  →  spin DOWN (-½)                             │
│   Photon mid-transit      →  superposition of spin                      │
│                                                                         │
│   Measurement direction determines compression direction.               │
│   Compression direction determines which side photon lands on.          │
│   That side IS the measurement result.                                  │
│                                                                         │
│   ═══════════════════════════════════════════════════════════════════   │
│                                                                         │
│   THE PUMP CYCLE — FULLY EXPLAINED:                                     │
│                                                                         │
│   Step 1:  Photon at WHITE side                                         │
│            White side = HARD (flat, compressed)                         │
│            Membrane on black side = INFLATING                           │
│                                                                         │
│   Step 2:  Photon launches across coin                                  │
│            Mid-transit = superposition                                  │
│            Spring at maximum extension                                  │
│                                                                         │
│   Step 3:  Photon arrives at BLACK side                                 │
│            Black side = HARD (flat, compressed)                         │
│            Membrane on white side = INFLATING                           │
│                                                                         │
│   Step 4:  Photon launches back                                         │
│            Mid-transit = superposition again                            │
│            Repeat forever                                               │
│                                                                         │
│   Pump frequency = photon bounce frequency                              │
│   Bounce frequency = electron's internal clock                          │
│   Internal clock = local rate of time                                   │
│                                                                         │
│   ═══════════════════════════════════════════════════════════════════   │
│                                                                         │
│   THE SPRING:                                                           │
│                                                                         │
│   Same material as the disc — extended or compressed.                   │
│   Extended (soft) = photon in transit, coin open                        │
│   Compressed (hard) = photon at a side, coin closed                     │
│                                                                         │
│   The spring is the complex plane.                                      │
│   Its tension holds the photon's probability distribution.              │
│   |amplitude|² = spring tension² = Born rule.                           │
│   Probability of finding photon at a location                           │
│   = square of wave amplitude at that location.                          │
│   This is MECHANICAL, not postulated.                                   │
│                                                                         │
│   ═══════════════════════════════════════════════════════════════════   │
│                                                                         │
│   THE MEMBRANE:                                                         │
│                                                                         │
│   Always OPPOSITE color to the visible coin face.                       │
│   Driven by the trapped photon hitting each side.                       │
│   Phase state: water/gas/solid                                          │
│   (liquid = normal, gas = excited, solid = compressed/observed)         │
│                                                                         │
│   When photon hits white → membrane goes black                          │
│   When photon hits black → membrane goes white                          │
│   The membrane IS the visible signature of the photon's bounce.         │
│                                                                         │

│                                                                         │

│   THE ELECTRON IS A LIGHT CLOCK:                                        │
│                                                                         │
│   The trapped photon bouncing between two sides                         │
│   IS a light clock.                                                     │
│   Not analogous to a light clock — literally IS one.                    │
│                                                                         │
│   This means time dilation is not an analogy:                           │
│                                                                         │
│   Stationary electron:                                                  │
│   Photon bounces straight up and down.                                  │
│   Maximum bounce rate. Maximum internal time.                           │
│                                                                         │
│   Moving electron (velocity v):                                         │
│   Photon travels diagonal path (longer).                                │
│   Fewer bounces per external second.                                    │
│   Slower internal clock.                                                │
│   THIS IS TIME DILATION — literal mechanism.                            │
│                                                                         │
│   v_internal = √(c² - v²)                                               │
│   Bounce rate = f₀ · √(1 - v²/c²) = f₀/γ                                │

│                                                                         │

│   → position unknown                                                    │
