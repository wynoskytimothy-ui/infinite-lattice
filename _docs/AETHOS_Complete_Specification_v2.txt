1. Executive Summary & Core Principles

every positive integer to eight unique three-dimensional coordinates based on prime factorization.
This is not merely an indexing scheme—it is a fundamental geometric structure that reveals how
integers, primes, and space are intrinsically connected.
Core Properties:
 Eight-fold simultaneous existence: Every number occupies 8 positions at once, one along each
base vector
 Deterministic: Coordinates computed purely from prime factorization—no randomness, no
approximation
 Collision-free: No two distinct numbers share the same coordinate on any vector
 Invertible: Full reconstruction of prime factorization from any coordinate
 Infinitely scalable: Extends to arbitrary factorization complexity
 Geometrically consistent: Discrete coordinates with continuous closure—no contradictions
Critical Insight: This system doesn't just address numbers—it reveals the geometric
structure of primes and provides a foundation for anomaly detection, compression,

2. Foundational Concepts
2.1 Each Vector is Its Own 0→∞ Number Line
The system has 8 base vectors (v1 through v8), and each vector functions as an independent,
infinite number line extending from 0 to ∞. This is fundamentally different from a traditional
coordinate system where axes are shared.
Key Principle: When we say a number 'exists' on vector v1, we mean it occupies a specific
position along v1's infinite pathway. That same number simultaneously exists at different
positions on all 7 other vectors. Each vector tells a different geometric story about the same
number.
Example: The number 15 = 3 × 5 exists at 8 different coordinate positions:
 v1: (8, 0, 8)
 v2: (8, 0, -8)
 v3: (-8, 0, 8)
 v4: (-8, 0, -8)
 v5: (0, 8, 8)
 v6: (0, 8, -8)
 v7: (0, -8, 8)
 v8: (0, -8, -8)
Each of these 8 positions represents the number 15 from a different geometric perspective. They are
not 8 different numbers—they are 8 simultaneous manifestations of the same number in 8 different
directional pathways through 3D space.
2.2 Every Prime Becomes a New Origin
This is the most profound property of the system: Every prime number p creates a branching
point that acts as a new local origin for subsequent structure. We call these prime doors.
What happens at a prime door:
 The prime p anchors at position n=p along each of the 8 base vectors
 At this anchor point, 4 new "wings" branch off in sub-directions
 Each wing extends infinitely from the prime anchor
 The prime anchor becomes a local origin for all branches spawned from it
 This creates 8 base spines × 4 wings = 32 new pathways per prime
 Since primes are infinite, the branching hierarchy is infinite
Example: Prime 2 creates 32 branches, prime 3 creates 32 more branches (which can
branch at prime 2), prime 5 creates 32 more (which can branch at 2 and 3), and so on
infinitely.

6. Multi-Prime Branching (Recursive Structure)
When a number has multiple distinct prime factors p■ < p■ < ... < p■, the branching becomes
recursive. Each prime adds a new layer of structure.
6.1 The Prime Sum Invariant
For k distinct primes, define the cumulative prime sum:
S■ = p■ + p■ + ... + p■
The z-coordinate invariant is:
z■ = S■ + n
This invariant guarantees that the transgressor n can always be recovered:
n = z■ - S■
Example: For N = 2 × 3 × 5 with transgressor n = 1:
 S■ = 2 + 3 + 5 = 10
 z■ = 10 + 1 = 11
 Therefore: n = 11 - 10 = 1 ✓
6.2 Case Separation
For two primes a ≤ p with transgressor n, three cases exist:
 Case 1: n < a < p
 Case 2: a < n < p
 Case 3: a < p < n
Each case uses different coordinate formulas to maintain collision-freedom. The middle coordinate (y
in VA, x in VB) encodes which case applies:
 Case 1: |y| = a
 Case 2: |y| = n
 Case 3: |y| = p
